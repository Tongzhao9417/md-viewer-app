MD Viewer 下一版：文档旁边的 Agent

MD Viewer 的起点，是用更轻的方式打开目录、阅读 Markdown，而不必进入完整编辑器。 下一版会延续这个方向，把 Agent 放进同一个阅读工作区。

文档仍然在中心，辅助能力只隔着一个侧边栏。

理解当前文档的辅助

可以让 Agent 总结当前文件、解释某个段落、润色一段文字，或者为 README 准备说明文案。 Markdown 预览始终保持可见。

工作区仍然熟悉：左侧是文件浏览，中间是文档，右侧是对话。

仍然围绕 Markdown

主题、大纲、Mermaid 图表、标签页，以及 HTML / DOCX / PDF 导出仍然是一等能力。 Agent 会作为阅读器旁边的辅助，而不是替代阅读器本身。
