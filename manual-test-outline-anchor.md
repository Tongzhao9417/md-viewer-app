# 目录、层级与锚点跳转手测文档

这个文件用于测试这次修复的 3 个问题：

- 左侧 Document Outline 是否有正确的标题层级。
- 在编辑模式、分屏模式下，点击左侧 Document Outline 是否能定位到对应标题。
- 在预览内容里点击 Markdown 自带目录链接，是否能跳到对应标题。

## 手测步骤

1. 打开这个文件，观察左侧 Document Outline，确认 `四、核心模型` 下有 `4.1`、`4.2`、`TraceRecord`、`TraceNode` 等子层级。
2. 切换到 `编辑`，点击左侧 Document Outline 里的 `八、页面与消费模型`，确认编辑器滚动到对应标题行。
3. 切换到 `分屏`，点击左侧 Document Outline 里的 `九、API 设计`，确认左侧编辑器和右侧预览都滚动。
4. 切换到 `预览`，点击下面这个正文目录里的 `八、页面与消费模型`，确认预览内容能跳转。
